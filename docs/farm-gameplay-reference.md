# 王者荣耀农场 · 玩法参考文档

> 本文档整理农场核心玩法机制与数值规则。  
> 详细数值以附件 **农场数据.xls** 为准。  
> **说明**：各数据表仅收录当前已知的数值，表内最后一行并不代表游戏上限，后续版本可能继续扩展。

---

## 1. 核心资源

| 资源 | 说明 |
|------|------|
| **农场等级** | 初始 1 级（数据表当前记录至 77 级） |
| **经验值** | 通过收获、开垦、升级小摊等行为获得；达到升级门槛后方可升级 |
| **农场币** | 主要货币，用于升级、买种子、开垦土地、升级小摊等；出售作物获得 |

### 1.1 升级规则

- 每次升级需同时满足：**当前经验 ≥ 下一级所需经验**，且**支付对应农场币**。
- 升级成功后，**总经验减去本次升级所需经验**（经验可溢出保留，但须达到门槛才能再次升级）。
- 各等级升级费用、所需经验、解锁内容见 **农场等级表**。

## 2. 农田系统

### 2.1 基本规则

- 农场最多 **24 块农田**；1 级初始拥有 **3 块**。
- 随农场等级提升，可花费农场币 **开垦新农田**，并获得经验。
- 1 块农田同时只能种植 **1 颗作物**。

### 2.2 二级农田

- **24 块农田全部解锁后**，随等级提升可依次将农田升级为 **二级地**（土地表当前记录 16 块）。
- 升级二级地需消耗农场币，升级后获得经验。
- 二级地效果：**作物产量 +50%**（即产量 × 1.5）。

### 2.3 相关数据

- 开垦费用、获得经验、所需等级 → **土地表**

---

## 3. 作物系统

### 3.1 种植与收获

| 环节 | 规则 |
|------|------|
| 购买种子 | 每种作物有固定种子价格（农场币） |
| 生长 | 每种作物有固定收获时间 |
| 收获 | 获得固定产量与经验；作物进入仓库 |
| 出售 | 在仓库中出售作物换取农场币 |

### 3.2 基础总售价

**基础总售价**：作物在 **1 块普通农田**（非二级地）上成熟后，不考虑任何加成时获得的农场币总额。

- 不含：培育度加成、小摊加成、二级地产量加成、变异加成。
- 各作物属性（产量、总售价、经验、收获时间等）→ **农作物表**。

### 3.3 作物分类（按收获时间）

| 收获时间 | 说明 |
|----------|------|
| 30s / 2min / 5min / 20min | 早期短周期作物 |
| 1h / 8h / 16h / 32h | 中后期长周期作物 |

**12 级起解锁规律**：每次升级解锁的新作物，其收获时间在 **1h → 8h → 16h → 16h → 32h** 之间循环。

| 类型 | 识别方式 | 特点 |
|------|----------|------|
| **普通作物** | 常规收获时间 | 正常经验收益 |
| **高农场币作物** | 英雄作物前、收获时间为 16h | 金币收益高，但收获仅 **1 经验** |
| **英雄作物** | 收获时间为 32h | 可发生英雄变异；极少数作物带指定英雄（见农作物表「变异英雄」列） |

### 3.4 相关数据

- 解锁等级、种子价、产量、总售价、经验、收获时间、各作物变异数量上限 → **农作物表**（已收录作物见表内条目）

## 4. 培育度系统

### 4.1 基本规则

- 每收获 **1 颗作物**，该作物种类获得 **10 点培育度**。
- 每种作物初始 **1 级**，售价加成为 **0%**。
- 消耗培育度可提升作物等级；**满级售价加成 100%**。
- 各等级所需培育度 → **培育度表**（「合计」列为累计所需，2～10 级列为各级增量）。

### 4.2 特殊作物规则

培育度表「备注」列标注了部分作物的非标准加成，例如：

- 部分早期作物（小麦、胡萝卜、番茄等）：2～3 级有固定加成比例。
- 部分作物（蘑菇、土豆、玉米等）：每升 1 级售价额外提升 20%。
- 备注为空的作物：1～8 级每升 1 级售价额外 +10%，9 级升 10 级额外 +20%。

## 5. 变异系统

### 5.1 触发时机

作物在以下环节均可能变异：

- 生长、浇水、成熟
- 被好友祝福、被好友拿取

### 5.2 变异类型与倍率

| 品质 | 变异类型 | 倍率 | 适用作物 |
|------|----------|------|----------|
| 蓝色 | 绿光、桃霞、幽蓝、橙光 | ×1.5 | 全部 |
| 紫色 | 铸铁、冰晶、青铜、琥珀 | ×2.5 | **非英雄作物** |
| 紫色 | 琉璃、青玉、明珠、璀璨 | ×2.5 | **英雄作物** |
| 黄色 | 巨大化 | ×8 | 全部 |
| 彩色 | 英雄 | ×28 | 全部（仅极少数作物可触发） |

- 各变异解锁等级 → **变异倍率表**、**农场等级表**（解锁内容）
- 每种作物可叠加 **不同类型** 的变异；**相同颜色词条不可叠加** 到同一作物上。
- 倍率计算方式：**乘算**（例：蓝色 ×1.5 + 紫色 ×2.5 → 1.5 × 2.5 = 3.75 倍）。
- 每种作物有 **变异数量上限**，各作物具体数值见 **农作物表「变异上限」列**。

## 6. 好友互动

### 6.1 好友祝福

| 规则 | 数值 |
|------|------|
| 解锁条件 | 农场等级 ≥ 4 |
| 每日祝福次数（主动） | 10 次 |
| 单颗作物最多被祝福 | 3 次 |
| 单农场每日最多被祝福 | 30 次 |
| 已达变异上限的作物 | 不可被祝福 |
| 刷新时间 | 每日 **凌晨 5:00** |

祝福有概率使作物发生变异。

### 6.2 作物偷取（拿取）

| 规则 | 数值 |
|------|------|
| 解锁条件 | 农场等级 ≥ 4 |
| 每日可拿取总量 | 120 颗（跨好友累计） |
| 单颗作物最多被拿取 | 4 次（4 个不同好友） |
| 被拿取后剩余产量 | 最多被拿走 40%，剩余 **60%** |
| 每次拿取量 | 相对平均分配，**不一定恰好为产量的 10%** |
| 刷新时间 | 每日 **凌晨 5:00** |

## 7. 浇水机制

### 7.1 水分维持度

设作物标准收获时间为 **T**：

| 概念 | 公式 / 说明 |
|------|-------------|
| 初始状态 | 刚种下时水分维持度为空（**干涸**） |
| 满水维持时间 **W** | W = T / 3 |
| 浇水效果 | 减少收获时间 = (W − w) / 4，其中 w 为当前水分维持度 |
| 消耗水分 **C** | C = W − w |
| 浇水门槛 | 仅当 **C ≥ T / 30** 时方可浇水 |
| 单次浇水 | 使水分维持度变为满 |

### 7.2 常见浇水策略

**勤奋策略**（共减少 T/4 收获时间）：

1. 刚种下浇 1 次 → 减少 T/12
2. 等干涸后浇 1 次 → 减少 T/12
3. 再等干涸后浇 1 次 → 减少 T/12
4. 剩余时间自然成熟

**极限策略**（共减少 4T/15 收获时间）：

1. 刚种下浇 1 次 → 减少 T/12
2. 等干涸后浇 2 次（各减少 T/12）
3. 再等待 T/15 后浇 1 次 → **直接成熟**

### 7.3 相关解锁

- 4 级解锁 **一键浇水**；8 级解锁 **一键收获**（见农场等级表）。

## 8. 农场小摊

### 8.1 解锁与升级

| 规则 | 说明 |
|------|------|
| 解锁条件 | 农场等级 ≥ 5 |
| 初始等级 | 1 级（出售价格加成 0%） |
| 可升等级 | 随农场等级提升而提高（见农场等级表、小摊表；当前数据未必覆盖全部等级） |
| 每升 1 级 | 出售价格加成 **+5%** |
| 升级消耗 | 农场币；升级后获得经验 |

**加成示例**：小摊 5 级 → 加成 20% → 出售价为原价的 **1.2 倍**。

公式：**小摊售价加成 = (小摊等级 − 1) × 5%**

### 8.2 相关数据

- 升级费用、获得经验、所需农场等级、各级加成 → **小摊表**

---

## 9. 周末双倍产量

- **时间窗口**：每周 **周五 18:00 ～ 周日 24:00**。
- **效果**：在此期间 **成熟** 的作物，产量变为原来的 **2 倍**。
- 仅影响产量，不改变单颗售价加成逻辑。

## 11. 售价与收益计算

### 11.1 实际出售单价（单颗作物）

```
实际单价 = 基础单价 × (1 + 培育度售价加成) × (1 + 小摊售价加成) × 变异倍率乘积
```

其中：

```
基础单价 = 基础总售价 / 基础产量
培育度售价加成 = 当前作物等级对应加成（0% ~ 100%，见培育度表）
小摊售价加成 = (小摊等级 − 1) × 5%
变异倍率乘积 = 各已触发变异倍率相乘（同品质不可叠加）
```

### 11.2 单次收获总收益

```
实际产量 = 基础产量 × 二级地系数 × 周末双倍系数
```

| 系数 | 取值 |
|------|------|
| 二级地系数 | 普通地 1.0；二级地 1.5 |
| 周末双倍系数 | 非周末窗口 1.0；周末窗口内成熟 2.0 |

```
单次总收益 = 实际单价 × 实际产量
```

> 被好友拿取后，剩余 60% 产量归农场主。

### 11.3 经验获取

- 普通作物：按 **农作物表「经验」列**（通常为整株收获经验）。
- 高农场币作物（16h、英雄作物前）：固定 **1 经验**。
- 开垦土地、升级小摊等额外经验见对应数据表。

## 13. 术语对照

| 术语 | 含义 |
|------|------|
| 农场币 | 游戏内主要货币 |
| 培育度 | 按作物种类累计，用于提升该作物售价加成 |
| 基础总售价 | 无加成条件下，1 块普通农田单次收获的总金币 |
| 高农场币作物 | 16h 收获、经验仅 1 的非英雄高价作物 |
| 英雄作物 | 32h 收获，可触发英雄变异的特殊作物 |
| 二级地 | 产量 +50% 的升级农田 |
| 小摊 | 5 级解锁，提供全局出售价格百分比加成 |
